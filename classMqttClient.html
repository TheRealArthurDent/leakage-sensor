Detailed Description

A very basic MQTT client.

This fella just showcases the use of marvinroger's Async MQTT client for ESP8266 and ESP32. Extend it to build more sophistcated stuff.

This method will be called by WiFiConnection once a network connection has been established. It will then try to connect to the configured MQTT broker using marvinroger's Async MQTT client for ESP8266 and ESP32.
